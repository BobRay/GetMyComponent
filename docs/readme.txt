
GetMyComponent
-----------
Author: Bob Ray <http://bobsguides.com>

MyComponent is an extensive example build script for MODx revolution
transport packages. It is designed so that you can modify it, and
use it to create your own transport packages.

This package, GetMyComponent, simply installs the MyComponent files.

Official Documentation: http://bobsguides.com/mycomponent-tutorial.html
Bugs and Feature Requests: https://github.com/BobRay/GetMyComponent
Questions: http://modxcms.com/forums
